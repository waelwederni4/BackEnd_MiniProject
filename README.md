## WebApi
# List all produits
`GET - /api/produits`
<br>
# Get produit by id
`GET - /api/produits/produit/:id`
<br>
# Get produits by nom
`GET - /api/produits/:nom`
<br>
# Create produit
`POST - /api/produits`
<br>
# Update produit
`PUT - /api/produits/:id`
<br>
# Delete produit
`DELETE - /api/produits/:id`
<br>
# Delete All produit
`DELETE - /api/produits`
